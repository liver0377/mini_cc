# Design
本文件描述整个项目工具系统的设计

## 工具设计
### 文件工具
- FileRead
  读取File, 目前仅支持文本文件
- FileEdit
  编辑文件，进行字符串替换操作
- FileWrite
  写入文件，进行全量文本写入操作


### Bash工具
- BashTool指一切可用的bash命令，需要进行严格的安全审核

### 搜索工具
下面两个工具均基于grep实现
- Glob
  按名称来搜索代码仓库中的文件
- Grep
  按名称来搜索代码文件中的片段

搜索到的结果可能会非常长，因此需要设定limit


## 工具注册
- 使用统一的类来描述所有的工具
- 使用工具注册表来注册所有的工具, 并且支持工具到API Format的转换
